# Old West Solutions - Enterprise IT Services Website

A modern, professional enterprise IT services website built with Next.js and TypeScript, featuring a dark Microsoft Windows theme.

## Features

- **Dark Windows Theme**: Authentic Microsoft Windows dark mode styling
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **TypeScript**: Fully typed for better development experience
- **Modern Stack**: Built with Next.js 14, React 18, and Tailwind CSS
- **Enterprise Focus**: Professional IT services presentation

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
